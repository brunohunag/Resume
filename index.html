個人簡介

我叫黃智彥，畢業於逢甲大學資訊工程系，主修網路工程。目前在緯創軟體擔任駐點網頁軟體工程師，駐點的公司為聯發科。
因小時後的我對電腦非常有興趣，喜歡研究有關電腦的硬體零件與維修。所以高中毅然決然念高職的資訊科，之後大學進入南台科技大學就讀資訊工程，但進入學校後教學狀況與技術方面不如我預期，也決定考轉學考到逢甲大學就讀。
而自己的個人特質為開朗樂觀、冒險、責任感重,遇到問題會希望能秉持著打破砂鍋問到底的精神，並且解決它。
平常時的興趣與休閒娛樂有慢跑、看影片、閱讀書籍、打保齡球。

為什麼想成為工程師

想當工程師是從小的夢想，小時候很喜歡碰有關電腦硬體的東西，組裝、維修(軟硬體都有)，但長大後從發現從事有關維修電腦的相關工作薪水不高，也就轉換了夢想，變改為全端工程師。
現在的自己有一些基礎在
前端技能:HTML,CSS,BootStrap,javascript,前端框架(vuejs)
後端技能:Node.js(Express),.NET Core
但希望能上到老師這麼門課程，是想要讓自己的技能樹能夠更深更廣
以下是我想學習的目標
1.加強自己在SQL方面的進階應用，過於複雜的查詢邏輯常常會不知道要如何撰寫
2.CSS也要加強，尤其是在切版方面，希望自己是能夠將css更加結構化撰寫(學會使用scss)
3.API Design,畢業後在撰寫API時，都沒有走正規的Restful API，並沒有清楚規劃GET,POST,PUT,DELETE
4.學會如何在專案上導入Scrum
5.CI/CD 技術因為從踏入職場後此技術是沒有碰過，但這個現在真的很多人都在使用
6.上完這門課程我想面試上這間公司，新加坡商鈦坦科技股份有限公司

為了成為工程師，曾經做過什麼努努力

那時剛畢業先從事手機驗證測試工作，但之後遇到一些工作上的不如意，讓我踏入網頁前後端的世界。
因此我整整花了兩個月專心做我的side project(記帳系統)，後段語言使用PHP（沒使用框架），
前端沒使用框架只有用jquery去做網頁互動與資料的前端處理，是一個網頁記帳系統那時對網頁不太熟，
很多都是先看別人的範例code改，雖然記帳系統並不是很完善，但基本的功能大致上都有，
登入/帳號申請/忘記密碼補發郵件/串接發票api/消費紀錄的新增修改刪除/金額統計。
也算是在那時候想踏入網頁前後端的工作時付出的努力，幸運的也讓我能找到相關工作。

如果參與這個計畫，會怎麼安排學習時間

因現在有工作，所以就是利用下班時間與週末分配
一～五的下班時間=>19:00~23:00
六日可以有16個小時的學習時間

對於工程師的職業想像是什麼？

就是能夠用自己學到的科學知識與技術，來幫助別人解決生活或商業上面的難題。
然後對於的電子軟硬體非常在行，很會修理東西。

如果參與這個計畫，預期會碰到什麼困難？你打算怎麼解決它？

通常學習有關電腦科學的相關知識容易碰到的就是無法將概念釐清，不太懂範例為什麼要這麼做？這個時候我想這樣做
1.先google看看別人有沒有跟你一樣不懂的，看高手怎麼幫他解答
2.與同儕討論，一起切搓
3.問老師，把你的想法跟學到的邏輯跟老師討論，讓老師指點你

想要對我們說的事情？

首先很謝謝老師能開這門課程，造福這麼多人，讓更多人成為軟體工程師
很希望彭彭老師能夠錄取我
我真的很渴望上這門課程
 或許我本身就是資工畢業但有關網頁技術都是靠自學，很多東西都算是一知半解
所以很希望能透過老師這門課
能讓我在這個領域更加進步
希望自己不要在原地踏步能從junior software engineer
變為 senior software engineer。
謝謝老師百忙之中閱讀我的履歷。
